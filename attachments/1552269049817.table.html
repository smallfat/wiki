Encrypt(
    plainText []byte, 
    address string) 
([]byte, error)
 使用用户地址为address的公钥对明文plainText进行加密

plainText: 明文数据 
address : 用户地址
2 
Decrypt(
    cipherText []byte,
    address string,
    password string)
([]byte, error)
 使用用户地址为address的私钥对密文cipherText进行解密

cipherText : 密文
address   : 用户地址
password  : 用户密码
3 
ReEncrypt(
    cipherText []byte, 
    address1 string,
   address2 string, 
    password string) 
([]byte, error)
 对已加密数据cipherText进行解密，然后再使用第二个用户地址的公钥进行加密。

cipherText: 密文数据 
address1 : 解密用户地址
address2 : 加密用户地址
password:  解密用户密码
